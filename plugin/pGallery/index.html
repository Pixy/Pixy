Exemples d'utilisation de pGallery

Exemples d'utilisation de pGallery.
Vous pouvez naviguer entre les différentes images grâce aux touches gauche et droite de votre clavier.
Pour fermer la gallerie, appuyez sur la touche Echap ou cliquez sur la croix en haut à droite.

EXEMPLE 1 (cliquez pour ouvrir la gallerie)
  • Une langue
  • Guizmo ?
  • Music
  • Deadmau5
  • Nike
  • RATP
  • Un plus long titre pour tester comment ça fonctionne

Le code :

$('#pGallery').pGallery({
'thumbnails' true, // Affichage ou non de la navigation par Thumbnails
'navigation': true, // Affichage ou non de la navigation par flèches
'currentImg': 0, // Première image appelée
'fadeTime': 500, // Durée de la transition lors de l'animation
'circular': true, // Circulaire
'auto': false, // Navigation automatique
'interval': 5000, // Intervalle pour la navigation automatique
'title': 'Titre principale du slider', // Titre principale du Slider
'keyboardNav': true // Navigation par les flèches du clavier
'caption': false // Affichage des captions
});
